## GET /health

Check API availability.

<Warning>
  The latest OpenAPI contract applies global security at the document level, but `/health` is commonly
  used as a simple liveness check. Confirm whether your environment requires authentication before relying
  on anonymous access to this endpoint.
</Warning>

<ParamField query="none" type="none">
  No parameters required.
</ParamField>

### Response

<ResponseField name="status" type="string" required>
  Always `"ok"` when the API is healthy.
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET "https://ramp.hexarails.ai/health" \
  -H "Content-Type: application/json"
```

```json Response
{
  "status": "SUCCESS",
  "data": {
    "status": "ok"
  }
}
```
</CodeGroup>

## POST /callback-url

Update the webhook callback URL.

### Request Body

<ParamField body="callbackUrl" type="string" required>
  HTTPS URL to receive webhook events. Must be reachable from Coinut servers.
</ParamField>

### Response

<CodeGroup>
```bash cURL
curl -X POST "https://ramp.hexarails.ai/callback-url" \
  -H "Content-Type: application/json" \
  -H "X-API-Key: $API_KEY" \
  -H "X-Timestamp: $TIMESTAMP" \
  -H "X-Nonce: $NONCE" \
  -H "X-Signature: $SIGNATURE" \
  -d '{"callbackUrl":"https://your-domain.com/webhooks/coinut"}'
```

```javascript JavaScript
authedRequest('POST', '/callback-url', {
  callbackUrl: 'https://your-domain.com/webhooks/coinut'
}).then(console.log);
```

```python Python
print(authed_request('POST', '/callback-url', {
    "callbackUrl": "https://your-domain.com/webhooks/coinut"
}))
```

```typescript TypeScript
authedRequest('POST', '/callback-url', {
  callbackUrl: 'https://your-domain.com/webhooks/coinut'
}).then(console.log);
```
</CodeGroup>

```json Response
{
  "status": "SUCCESS",
  "data": {
    "message": "Callback URL updated successfully"
  }
}
```

<Note>
  The callback URL must use HTTPS. Coinut verifies reachability before saving.
</Note>

## POST /ip-white-list

Update the IP whitelist.

### Request Body

<ParamField body="ipWhiteList" type="string[]" required>
  Array of IPv4 or IPv6 addresses to whitelist. Replaces the existing list entirely.
</ParamField>

<CodeGroup>
```bash cURL
curl -X POST "https://ramp.hexarails.ai/ip-white-list" \
  -H "Content-Type: application/json" \
  -H "X-API-Key: $API_KEY" \
  -H "X-Timestamp: $TIMESTAMP" \
  -H "X-Nonce: $NONCE" \
  -H "X-Signature: $SIGNATURE" \
  -d '{"ipWhiteList":["1.2.3.4"]}'
```

```javascript JavaScript
authedRequest('POST', '/ip-white-list', {
  ipWhiteList: ['1.2.3.4']
}).then(console.log);
```

```python Python
print(authed_request('POST', '/ip-white-list', {
    "ipWhiteList": ["1.2.3.4"]
}))
```

```typescript TypeScript
authedRequest('POST', '/ip-white-list', {
  ipWhiteList: ['1.2.3.4']
}).then(console.log);
```
</CodeGroup>

```json Response
{
  "status": "SUCCESS",
  "data": {
    "message": "IP whitelist updated successfully"
  }
}
```

<Warning>
  This endpoint replaces the entire whitelist. Include all desired IPs in every request. Omitting an address removes it.
</Warning>
