## Get Payment Estimate

Preview the exchange rate and fees before creating a payment.

   

### Request

<CodeGroup>

```bash
curl -X GET "https://partner.coinut.net/api/payment/estimate?deductCurrency=USDT&amount=100&currency=USD&bankAccountId=ba-550e8400" \
  -H "X-COINUT-PARTNER-APIKEY: your_api_key" \
  -H "X-COINUT-PARTNER-NONCE: 1710825600000" \
  -H "X-COINUT-PARTNER-SIGNATURE: your_signature"
```

```javascript
const params = new URLSearchParams({
  deductCurrency: "USDT",
  amount: "100",
  currency: "USD",
  bankAccountId: "ba-550e8400",
});

const response = await fetch(`https://partner.coinut.net/api/payment/estimate?${params}`, {
  method: "GET",
  headers: {
    "X-COINUT-PARTNER-APIKEY": "your_api_key",
    "X-COINUT-PARTNER-NONCE": "1710825600000",
    "X-COINUT-PARTNER-SIGNATURE": "your_signature",
  },
});

const data = await response.json();
```

## Create Payment

Initiate a fiat payment from your partner balance.

<Warning>
  Ensure your partner balance has sufficient funds before creating a payment. The `deductAmount` returned by the estimate endpoint will be debited from your balance.
</Warning>

<Note>
  The `supportingDocument` field is optional. Attach an invoice, purchase order, or compliance document when required.
</Note>

     

### Request

<CodeGroup>

```bash
curl -X POST "https://partner.coinut.net/api/payment/create" \
  -H "X-COINUT-PARTNER-APIKEY: your_api_key" \
  -H "X-COINUT-PARTNER-NONCE: 1710825600000" \
  -H "X-COINUT-PARTNER-SIGNATURE: your_signature" \
  -H "Content-Type: application/json" \
  -d '{
    "deductCurrency": "USDT",
    "amount": "100",
    "currency": "USD",
    "bankAccountId": "ba-550e8400",
    "reference": "payment-123456"
  }'
```

```javascript
const response = await fetch("https://partner.coinut.net/api/payment/create", {
  method: "POST",
  headers: {
    "X-COINUT-PARTNER-APIKEY": "your_api_key",
    "X-COINUT-PARTNER-NONCE": "1710825600000",
    "X-COINUT-PARTNER-SIGNATURE": "your_signature",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    deductCurrency: "USDT",
    amount: "100",
    currency: "USD",
    bankAccountId: "ba-550e8400",
    reference: "payment-123456",
  }),
});

const data = await response.json();
```

## Payment Detail Schema

### GetPaymentDetailDataDto

| Field | Type | Description |
| --- | --- | --- |
| `id` | `string` | Payment UUID |
| `status` | `string` | `PROCESSING` or `CONFIRMED` |
| `rate` | `string` | Exchange rate applied |
| `deductAmount` | `string` | Amount deducted from partner balance |
| `deductCurrency` | `string` | Currency deducted |
| `receivedCurrency` | `string` | Currency received by beneficiary |
| `receivedAmount` | `string` | Amount received |
| `bankCharge` | `string` | Bank fee charged |
| `reference` | `string` | Payment reference |
| `createTime` | `string` | Creation timestamp |
| `bankAccount` | `object` | Beneficiary bank account details |

### PaymentDetailBankAccountDto

| Field | Type | Description |
| --- | --- | --- |
| `bankName` | `string` | Bank name |
| `accountNo` | `string` | Account number |
| `bankCountry` | `string` | Bank country code |
| `bankAddress` | `string` | Bank street address |
| `swift` | `string` | SWIFT/BIC code |
| `beneficiaryName` | `string` | Account holder name |
| `beneficiaryAddress` | `string` | Account holder address |
